Other projects

Chess

Over the summer of 2020, I started working on a chess game, written using JavaFX. It's a work in progress, although most functionality is already implemented.
Once finished, features should include:

  • LAN multiplayer
  • Archiving of completed games plus replay functionality
  • Move highlighting
  • Custom timing and rules
  • Saving/loading of in-progress games
  • Spectating
  • Switching between 2D and 3D views

And then I might try to implement an AI opponent, perhaps using machine learning.


Theorem prover

I've also been working collaboratively with a couple of others on developing an interface for, and corresponding implementation of, the theorem prover Nuprl, or computational type theory.

So far, I've implemented the primary features of a frontend of the interface, with standard text editor features and input and output text areas. Work currently is focussing on the underlying logic, where we parse the input, evaluate the terms and display an output to the user. We're hoping to at least have a baseline that, although feature-incomplete, can be expanded on and improved on as time goes on until it develops into a good, useable standalone interface.
